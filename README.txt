- сайт подготовлен для деплоя на Vercel без серверной части.

Запуск локально:
1. Откройте index.html в браузере.
2. Для просмотра справочника РД откройте rd.html или ссылку в задачах.

Коды входа по умолчанию:
- сотрудники: 1234
- администратор: 0000

Деплой на Vercel через GitHub:
1. Распакуйте архив.
2. Загрузите содержимое папки сайта в корень репозитория GitHub.
3. В Vercel создайте новый проект из репозитория.
4. Application Preset: Other.
5. Build Command: пусто или echo static site.
6. Output Directory: .
7. Deploy.

Файлы в корне репозитория должны лежать так:
index.html
rd.html
vercel.json
package.json
css/
js/
