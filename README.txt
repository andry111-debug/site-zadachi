Локальный статический сайт задач сотрудников. Версия в вечерней темной цветовой схеме.

Запуск локально:
1. Откройте index.html в браузере.
2. Для просмотра справочника РД откройте rd.html или ссылку в верхнем меню.

Коды входа по умолчанию:
- сотрудники: 1234
- администратор: 0000

Для деплоя на Vercel:
1. Зайдите на https://vercel.com и создайте новый проект.
2. Загрузите содержимое этой папки в репозиторий GitHub/GitLab/Bitbucket или импортируйте как статический проект.
3. Build Command можно оставить пустым или указать: echo static site
4. Output Directory оставьте пустым / корневой каталог проекта.
5. После деплоя главная страница будет index.html, справочник РД — rd.html.

Сайт без серверной части. Отметки выполнения сохраняются в браузере пользователя через localStorage.


Вечерняя схема:
- темный сине-графитовый фон;
- светлый текст;
- янтарный акцент для ссылок и кнопок;
- сохранена мобильная адаптация и статический формат для Vercel.
